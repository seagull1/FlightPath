GPS for small aircraft polit

Flight Path optimization solution for small aircraft pilot. Small aircraft might need make some stops in order to reach final distance. This solution provides shortest flying route between departure and arrival airport. The route is a list of airport, landing and take off along the list help reduce flying time.

API

For example if fly from Vancouver to NY with a fly range 400 miles small aircraft, some stops are needed for fuel. This api can generate an optimized route in order to arrive NY by a shortest flight path.

Code on GitHub

Http Verb URL Description Example
GET /api/airport/{designator} Get airport by designator api/airport/CYVR
GET api/aircraft/{Id} Get an aircraft by Id api/aircraft/1
GET api/aircraft/{make}/{model} Get list of aircrafts by make and model api/aircraft/aeronca/champ
